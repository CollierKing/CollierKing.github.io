Welcome

This site explores real-world problems with data science, machine learning and other statistical and quantitative analysis techniques. These projects are grouped into four categories.

Machine Learning

Trains classification and regression models on real-world datasets gathered from web-scraping and APIs to make practical decisions. Project code is written with Python and hosted via Jupyter Notebook.


Marketing Analytics

Applies unsupervised-learning and classical statistical techniques to learn about consumer interests and economic patterns. Data munging and modeling is carried out in R and Python.


Business Modeling

Explores quantitative techniques aimed at solving business optimization problems, such as pricing and risk management. Exercises are written in Python.


Apps & Dashboards

Contains dashboards and applications written in R Shiny and Python Dash for stock market analysis and statistical research.
